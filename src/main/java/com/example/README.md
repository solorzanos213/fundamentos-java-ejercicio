# Fundamentos de Lógica en Java

Este proyecto contiene ejemplos de conceptos básicos de programación en Java.

## Estructura

src/main/java/com/example

## Temas implementados

- Variables y constantes
- Tipos de datos primitivos
- Operadores
- Condicionales
- Ciclos
- Arreglos
- Cadenas

## Cómo ejecutar

Compilar:
javac src/main/java/com/example/*.java

Ejecutar:
java -cp src/main/java com.example.App
